# Projet Web - Site EFREI

Bonjour ! Bienvenue sur le dépôt de notre projet de site web pour le département Informatique de l'EFREI.

**Projet réalisé par :** Ali Chaoui El Faiz, Wacim Hamdi Aissa et Samy Aouchiche.

## À propos du contenu
Pour que le site soit plus réaliste et agréable à parcourir, la majorité des textes (actualités, descriptions, FAQ...) a été inspirée de vrais sites, de forums ou générée par IA. Le but était d'améliorer l'expérience utilisateur et d'éviter les pages vides et maigre d'informations.

## Pourquoi des fichiers "ANCIEN" 
En fouillant dans les dossiers, vous croiserez des fichiers avec "ANCIEN" dans leur nom. Pendant le développement du code nous avons fait plusieurs tests et nous avons changé le style visuel à plusieurs reprises. Nous avons gardé ces anciennes versions de côté au cas ou.

## Historique Git et Live Share
Vous remarquerez peut-être qu'il y a relativement peu de commits sur ce dépôt. C'est parce que nous avons réalisé la grande majorité du projet en travaillant en simultané grâce à "Live Share" de VS Code. Le code a donc été écrit à plusieurs en temps réel plutôt que via des pushs séparés réguliers.

## Fonctionnalités principales
* Design Responsive : Le site s'affiche sur PC comme sur mobile.
* Carrousel d'images : Défilement dynamique sur la page d'accueil.
* Chatbot intégré : Un chatbot codé en JS pour répondre aux questions.
* Carte interactive : Intégration d'une carte Google Maps sur la page Contact.

## Structure des dossiers
* /html : Les différentes pages du site (campus, contact, equipe...).
* /css : Toutes les styles css triées par page.
* /js : Les scripts pour rendre le site dynamique.
* /img : Les photos et logos utilisés.

## Améliorations possibles
Si nous avions eu plus de temps, voici ce que nous aurions aimé ajouter :
* Un vrai backend (PHP ou base de données) pour rendre le formulaire de contact 100% fonctionnel.
* Un mode sombre (Dark Mode).
* Des animations d'apparition au défilement de la page.
